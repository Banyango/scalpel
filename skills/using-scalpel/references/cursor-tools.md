# Cursor Tool Mapping

Skills speak in actions. On Cursor these resolve to the tools below.

| Action | Cursor tool |
|---|---|
| Read a file | `read_file` |
| Create / edit a file | `edit_file` |
| Run a shell command | `run_terminal_cmd` |
| Search file contents | `grep_search` |
| Find files by name | `file_search` |
| Fetch a URL | `web_search` |
| Invoke a skill | `skill` |
| Dispatch a subagent | `agent` |
| Task tracking | `add_task`, `update_task` |

## Instructions File

When a skill mentions "your instructions file": on Cursor this is `.cursorrules` or `.cursor/rules/*.mdc`.
