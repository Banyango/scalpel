## Scalpel Plugin — Codex / Copilot CLI

This project is the Scalpel skills plugin itself. When working in this repo:

- Skills live in `skills/<name>/SKILL.md`
- Plugin manifests live in `.<provider>-plugin/plugin.json`
- The entry skill is `skills/using-scalpel/SKILL.md`

When adding a new skill, create `skills/<name>/SKILL.md` with `name` and `description` frontmatter. Update the Available Skills list in `skills/using-scalpel/SKILL.md`.
